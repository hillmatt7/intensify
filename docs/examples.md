# Examples

The `examples/` directory contains domain-oriented notebooks. They are written
as starting points for adapting intensify to real datasets rather than as API
reference material.

- `examples/neuroscience_spike_analysis.ipynb`: spike-train loading,
  exploratory summaries, Hawkes fitting, and diagnostics.
- `examples/seismology_aftershocks.ipynb`: aftershock clustering with
  self-exciting event dynamics.
- `examples/biology_epidemiology_ecology.ipynb`: event models for biological
  and ecological count processes.

For a text-first path through the same ideas, see the domain guides:

- [Quantitative finance](user_guide/finance.md)
- [Computational neuroscience](user_guide/neuroscience.md)
